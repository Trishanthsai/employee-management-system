# Employee Management System (Spring Boot CRUD)

A simple Employee Management System built using Spring Boot + Spring Data JPA.

## Features
- Add Employee
- Get All Employees
- Get Employee By ID
- Update Employee
- Delete Employee
- Validation using 
- Global Exception Handling 
- Custom Exception: Employee Not Found (404)
- ResponseEntity with proper HTTP status codes
- Pagination + Sorting

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Swagger UI
- Postman

## API Endpoints
- GET /employee
- GET /employee/{id}
- POST /employee
- PUT /employee/{id}
- DELETE /employee/{id}

## How to Run
1. Clone the repo
2. Configure DB in `application.properties`
3. Run the project
4. Open Swagger:
   http://localhost:8080/swagger-ui/index.html
